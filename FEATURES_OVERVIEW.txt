Field Name:        github_profile_url
Type:              String (URL)
Required:          No (optional)
Example Value:     https://github.com/your-username
Display:           Clickable GitHub username link on prompt card
Icon:              GitHub icon (fab fa-github)
Fallback:          "System" with robot icon if not provided
JSON Location:     Add at the end of prompt JSON object


EXAMPLE PROMPT JSON:

{
  "id": 1,
  "title": "Expert Code Reviewer",
  "role": "Principal Software Engineer",
  "task": "Review code for issues",
  ...other fields...,
  "github_profile_url": "https://github.com/your-username"
}
